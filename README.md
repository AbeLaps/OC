# OC

## Projeto da disciplina de Organização de Computadores
# 1 Fase do projeto
A primeira fase consistiu na criação de um computador de 16 bits com a assistencia do software LogSim.

# 2 Fase do projeto
A segunda fase consistiu na codificação de um montador como um compilador com base em uma linguagem assembly inspirada em uma arquitetura x86 adaptada.

O montador compila um arquivo assembly para um arquivo compatível com a importação dos dados nativa do bloco de memória do LogSim
